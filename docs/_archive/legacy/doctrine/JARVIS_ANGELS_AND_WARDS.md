# Jarvis Angels And Wards

The Angels and Wards doctrine is a live evaluation layer for Jarvis modular
assembly.

Angels:

- `SHIELD`: preserve core identity
- `PROTECT`: preserve structural stability
- `GUARD`: preserve subsystem boundaries

Wards:

- `WARD`: prevent contamination between context layers
- `SEE`: require inspectability and traceability
- `WEARY`: detect fatigue, repetition, and degraded loops
