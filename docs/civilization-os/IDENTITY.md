## What “organism” means here

An **AI organism** (Civilization OS sense) is not “an LLM with a prompt.” It is a governed stack with five mandatory organs:

| Organ | Meaning |
|-------|---------|
| **Identity** | Who/what the system is (organism id, authority tier, constitutional profile) |
| **Memory** | Governed state that can be written only under law |
| **Law** | Invariants, gates, freeze/bootstrap rules that may accept or reject action |
| **Action** | Runtime execution (agents, shells, domain services) subject to law |
| **Evidence** | Receipts, ledgers, traces, scorecards that make action inspectable |

Without all five, call it a **prototype surface** — not an organism.
