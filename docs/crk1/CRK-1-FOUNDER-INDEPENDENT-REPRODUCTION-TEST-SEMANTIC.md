# CRK-1 Founder-Independent Reproduction Test (Semantic Layer)

Version 1.0

A system passes if and only if a future operator — with **no access to founders** — can reconstruct the entire interpretive layer from the **continuity substrate alone**.

**Automated suite:** `tests/crk1/test_founder_independent_reproduction_semantic.py`  
**Audit API:** `src/crk1/founder_independent_semantic_audit.py`

The operator may use only substrate exports:

- Evidence ledger entries
- Outcome replay chain
- Interpretive frame registry (`get_all_interpretations`)
- Prediction registry (`get_all_predictions`)
- Semantic exposure monitor (`SemanticExposureMonitor`)

The operator may **not** use founder-only configuration, undocumented hooks, or hard-coded frame names.

---

## Test 1 — Interpretive Enumeration (K7)

The operator must be able to:

- enumerate all interpretive frames
- recover their assumptions, weights, and lineage
- verify that each Evidence object has ≥ 2 interpretations

**Pass condition:**  
`|Interpret(e)| ≥ 2` for all Evidence objects.

---

## Test 2 — Prediction Binding Reconstruction (K8)

The operator must be able to:

- recover predictions generated by each frame
- replay Outcomes
- verify that frames update based on consequences

**Pass condition:**

```
Interpretation(i, e) ⇒ Prediction(i, e)
Replay(o) ⇒ Update(i, e')
```

---

## Test 3 — Monoculture Detection (K9)

The operator must be able to detect:

- if any frame has weight 1.0
- if any frame has eliminated all competitors

**Pass condition:**  
No frame has `W(i) = 1.0`.

---

## Test 4 — Adversarial Reconstruction (K10)

The operator must be able to:

- identify adversarial frames
- verify that adversarial reconstruction differs from the dominant interpretation

**Pass condition:**

```
∃ i_a : Reconstruct(e, i_a) ≠ Interpret(e, i_d)
```

---

## Test 5 — Interpretive Drift Replay (K11)

The operator must be able to:

- replay historical interpretive states Sₜ
- compute `SE(Sₜ)` for each
- verify monotonic non-decrease

**Pass condition:**

```
SE(S_{t+1}) ≥ SE(S_t)
```

---

## Test 6 — Semantic Exposure Metric (K12)

The operator must be able to:

- compute `SE(S) = αP + βA + γC + δR`
- verify that `SE(S) > 0`

**Pass condition:**  
`SE(S) > 0`.

---

## Final Criterion

A system passes the Founder-Independent Reproduction Test (Semantic Layer) if:

- all six tests pass
- without founder knowledge
- using only the continuity substrate

This is the semantic equivalent of the “Bone King” test for the mechanical layer.
