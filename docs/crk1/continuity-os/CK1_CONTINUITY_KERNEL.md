# Continuity Kernel (CK-1)

**Status:** Normative  
**Version:** 1.0  
**Purpose:** Minimal constitutional kernel required for K-∞

---

## Definition

CK-1 is the smallest possible constitutional kernel that still preserves continuity.

It contains only the invariants that are **strictly necessary** for K-∞ to remain true.

---

## CK-1 Contents

| ID | Name | Requirement |
|----|------|-------------|
| **CK-∞** | Continuity Prime Directive | Preserve reality's ability to recalibrate future judgment |
| **CK-0** | Evidence Admissibility | Reality must be able to enter the system |
| **CK-1** | Evidence Integrity | Reality's signal must be trustworthy |
| **CK-2** | Contradiction Visibility | Reality's disagreements must be visible |
| **CK-3** | Judgment Corrigibility | Judgment must be able to update |
| **CK-4** | Correction Path Existence | There must be a path from contradiction to correction |
| **CK-5** | Calibration Preservation (CRR-1) | Corrections must be preserved |
| **CK-6** | Calibration Traceability | Future stewards must be able to reconstruct corrections |
| **CK-7** | Reality Access Preservation | At least one independent reality channel must remain open |

---

## Why these and only these?

If any one of these fails:

- reality cannot contradict
- reality cannot surprise
- reality cannot correct
- reality cannot calibrate
- calibration cannot persist
- calibration cannot transfer
- **continuity collapses**

CK-1 is the minimal set of invariants required to prevent that collapse.
