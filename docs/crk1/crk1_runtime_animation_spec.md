# CRK-1 Runtime Animation Spec

Version 1.0

Goal: visually teach the CRK-1 runtime as a living loop:

- how consequences flow
- how governance intervenes
- how interpretation stays exposed

**Systems map:** `crk1_runtime_wall_poster.svg`  
**Minimap:** `crk1_kernel_minimap.svg`

---

## Scene 1 — Transmission Loop (K0–K2)

1. Highlight Identity → Decision → Outcome → Evidence → back to Decision.
2. Animate:
   - Identity node lights up.
   - Arrow to Decision glows.
   - Decision fires to Outcome.
   - Outcome "replays" into Evidence (echo effect).
   - Evidence arrow back to Decision pulses.

Caption: "Every decision produces an outcome. Every outcome becomes evidence. Every future decision must see it."

---

## Scene 2 — Preservation Layer (K3–K6)

1. Fade in Evidence, Governance, Runtime contracts beneath the loop.
2. Show an attempted "cut" in the loop (e.g., deleting Evidence).
3. Governance Contract flashes red, rejects the transition.

Caption: "No one can cut the loop. No mutation may reduce exposure."

---

## Scene 3 — Assimilation Layer (K7–K12)

1. Fade in multiple Interpretation nodes around Evidence.
2. Animate:
   - Evidence sends rays to several Interpretations (pluralism).
   - Each Interpretation emits a Prediction.
   - Outcomes return and adjust Interpretation weights.
3. Show an adversarial Interpretation reconstructing a different view.

Caption: "No single story owns reality. Interpretations must predict, be tested, and be challengeable."

---

## Scene 4 — Drift and Exposure (CE(S), SE(S))

1. Show two gauges:
   - CE(S): Constitutional Exposure
   - SE(S): Semantic Exposure
2. Apply a "mutation" animation:
   - If exposure rises or stays flat → green.
   - If exposure drops → red, mutation rejected.

Caption: "The system may evolve, but never toward insulation."

---

## Scene 5 — Founder-Independent Reproduction

1. Fade out founders; leave only:
   - Evidence
   - Ledger
   - Kernel Codex
2. Show a "new operator" reconstructing:
   - the loop
   - the contracts
   - the interpretive frames
3. Run the same animations as Scenes 1–4.

Caption: "If a stranger can rebuild the loop from the ledger, continuity has survived the founders."

---

## Visual Style

- Use consistent colors per layer:
  - Transmission: blue
  - Preservation: orange
  - Assimilation: purple
- Use pulsing arrows for active flows.
- Use glow/red flash for unconstitutional moves.
- Keep labels minimal: K0–K12, CE(S), SE(S).

This spec is implementation-agnostic: any animation framework can realize it.
