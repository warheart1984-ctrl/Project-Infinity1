# AAES-OS Constitution (v1.0)

Frozen constitutional principles for governed intelligence.

## Core principles

1. **Reality is the final authority** — claims require executable proof.
2. **No hidden state** — all behavior must be reconstructable from receipts.
3. **Deterministic execution** — identical inputs yield identical transitions.
4. **Constitutional boot** — invariants validated before runtime accepts work.
5. **Receipts or it didn't happen** — every governed action produces a content-addressed receipt.
