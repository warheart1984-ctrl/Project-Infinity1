## Decision Rules

| Change class | Rule |
|--------------|------|
| **Constitutional changes** | Supermajority of Foundational + Architectural stewards |
| **Runtime changes** | Architectural majority + no Foundational veto |
| **Implementation changes** | Implementation majority + Architectural sign-off |

## Continuity Validation

CDP-1 / CAA-1 continuity propagation claims require:

- ≥ 3 independent operational stewards (per [CDP1_MULTI_STEWARD_PROTOCOL.md](../../crk1/mission-006/CDP1_MULTI_STEWARD_PROTOCOL.md))
- ≥ 2 independent review stewards (per [MULTI_STEWARD_GOVERNANCE_CHARTER.md](../../crk1/governance/MULTI_STEWARD_GOVERNANCE_CHARTER.md))
- No substantiated proof-based veto from any review steward
