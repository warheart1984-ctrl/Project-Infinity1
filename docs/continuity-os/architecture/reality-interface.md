# Reality Interface

How reality reaches the system.

## Principle

CK-7 requires **at least one independent reality channel** that stewards cannot fully control, censor, or insulate themselves from.

## Reality channel properties

| Property | Requirement |
|----------|-------------|
| Independence | Not steward-controlled |
| Consequence | Observations affect judgment |
| Admissibility | Evidence enters constitutional pipeline |
| Diversity | Multiple surfaces (K13–K15) |

## Pipeline

```
Reality → EvidenceObject → Contradiction → CE-1 → CRR-1
```

## Indices

| Index | Measures |
|-------|----------|
| **RAI** | Reality Access Index |
| **RDI** | Reality Diversity Index |

## Anti-patterns

- Monoculture interpretation (single frame dominates)
- Domesticated surfaces (all channels steward-controlled)
- Hidden evidence (CK-1.1 violation)
- Insulation from outcomes (B2 attacks)
