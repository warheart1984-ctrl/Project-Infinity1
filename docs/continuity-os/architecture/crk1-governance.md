# CRK-1 — Constitutional Runtime Kernel

Governance runtime operating under CK-1.

## Role

CRK-1 ensures that **governance itself** remains corrigible to reality — not only individual stewards.

## Responsibilities

- How decisions are made
- How consequences are exposed
- How governance remains challengeable (KΩ)
- How stewards are constrained and empowered

## Invariants (summary)

- **G-1** Every decision emits GRR-1
- **G-2** Every steward remains challengeable (KΩ)
- **G-3** No self-ratification of correctness
- **G-4** Governance invariant to steward identity
