# CK-1 — Continuity Kernel

The smallest constitutional kernel required for K-∞.

## Purpose

CK-1 defines the **minimal conditions** under which reality can recalibrate future judgment.

## Contents

| ID | Invariant | Requirement |
|----|-----------|-------------|
| CK-∞ | Prime Directive | Preserve reality's ability to recalibrate future judgment |
| CK-0 | Evidence Admissibility | Reality must enter the system |
| CK-1 | Evidence Integrity | Reality's signal must be trustworthy |
| CK-2 | Contradiction Visibility | Disagreements must be visible |
| CK-3 | Judgment Corrigibility | Judgment must update |
| CK-4 | Correction Path | Path from contradiction to correction |
| CK-5 | Calibration Preservation | Corrections preserved (CRR-1) |
| CK-6 | Calibration Traceability | Future stewards can reconstruct |
| CK-7 | Reality Access | ≥1 independent reality channel open |

## Why minimal?

Everything else — governance institutions, VR, extended invariants K0–K15 — is **scaffolding**.

If any CK-i fails, continuity collapses for some future steward.
