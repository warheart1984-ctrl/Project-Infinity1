# Lineage Invariants (CLG-1)

Book of Invariants Sections IV–V (calibration + lineage).

## Calibration (CE-1)

| ID | Invariant |
|----|-----------|
| **C-1** | Contradiction must produce surprise |
| **C-2** | Surprise must produce correction |
| **C-3** | Correction must produce a preserved CRR-1 |
| **C-4** | Calibration deltas must be computable and comparable |

## Lineage (CLG-1)

| ID | Invariant |
|----|-----------|
| **L-1** | Every correction must enter lineage |
| **L-2** | Lineage must be reconstructible across stewards |
| **L-3** | No correction may be erased or overwritten |
| **L-4** | Lineage must remain independent of model architecture |

## CRR-1 vs GRR-1

| Receipt | Preserves |
|---------|-----------|
| GRR-1 | Judgment (why we thought) |
| CRR-1 | Corrigibility (why we stopped thinking) |

Continuity depends on CRR-1, not GRR-1 alone.
