# K-∞ — Continuity Prime Directive

## Statement

> The system must preserve reality's ability to recalibrate the judgment of future stewards.

## What K-∞ is not

K-∞ is not an invariant among others. It is the **constitutional purpose function** from which K0–K15, CK-1, and all artifacts derive.

## Corrigibility chain

| Mechanism | Role |
|-----------|------|
| Evidence | Reality can speak |
| Contradiction | Reality can disagree |
| Surprise | Reality can penetrate assumptions |
| Correction | Reality can change judgment |
| Calibration | Reality's lessons persist |
| Stewardship | Calibration transfers |
| Continuity | Corrigibility survives generations |

## Failure

When corrigibility collapses, continuity collapses — even if knowledge, institutions, and archives remain intact.

## Artifacts

**CRR-1** is the first concrete implementation of K-∞ — it preserves *why we stopped thinking this*.
