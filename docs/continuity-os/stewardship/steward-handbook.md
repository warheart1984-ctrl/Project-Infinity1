# Steward Handbook

Operator reference for lawful stewards.

## Daily workflow

1. **Emit expectation** — record what you predict, with confidence and assumptions
2. **Bind governance** — ensure GRR-1 context exists for the decision
3. **Receive evidence** — via independent reality channels only
4. **Detect contradiction** — never hide Δ between expectation and observation
5. **Accept correction** — run CE-1; do not self-ratify
6. **Preserve CRR-1** — every correction enters lineage
7. **Hand off** — successors must reconstruct your calibration

```bash
continuity demo falling-object    # practice MVCD
continuity mission 005          # multi-steward lineage

```

| Action | Violated invariant |

## Incident response

1. Identify which invariant failed
2. Consult [Collapse Taxonomy](../../crk1/continuity-os/CONTINUITY_COLLAPSE_TAXONOMY.md)
3. Emit corrective CRR-1 if reality already corrected judgment
4. Run red-team suite before resuming certification
