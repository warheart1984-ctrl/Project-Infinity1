# Bumblebee Forge Subsystem

This folder contains the governed planning artifacts for Bumblebee Forge Edition.

## Intent

Bumblebee Forge evolves scanner-first work into a governed reconstruction workflow.
The workflow is law-bound, proof-first, and safe-default.
