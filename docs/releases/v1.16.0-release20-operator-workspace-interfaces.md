# AAIS v1.16.0 — Release 20 Operator Workspace & Extended Interfaces

**Date:** 2026-06-03  
**Tag:** `v1.16.0`  
**Prior:** [v1.15.0 — Release 19 Operator Product Interfaces](./v1.15.0-alt19-operator-product-shell.md)
