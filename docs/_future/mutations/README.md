[AAIS_SUBSYSTEM_MUTATION_PATH.md](../../contracts/AAIS_SUBSYSTEM_MUTATION_PATH.md).

## Layout

```text
docs/_future/mutations/MP-<ID>_<gene>.md
schemas/deltas/<gene>_MP-<ID>.json   # optional schema delta
```
