# CEN Omega Stress Harness

## Purpose
Omega generates adversarial CEN scenarios for deterministic enforcement validation.

## Scenarios
- malformed payloads
- replay attacks
- threshold-skirt attempts
- high-frequency floods
- conflicting distributed writes
- partial trust and corrupted tokens

## Output
`runOmegaStressHarness()` returns scenario names, deterministic counts, and sovereignty ledger entries.
