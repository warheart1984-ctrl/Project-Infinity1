# Governed Artifact: Theta-BIP39 Profile v0.1

This directory registers Theta-BIP39 Encoding Profile v0.1 as a governed AAES
OS/NexusOS artifact.

The artifact binds:

- profile version
- canonical standards directory
- invariant requirements
- profile-critical hashes
- steward and change-control process
- state-vector and confidence-layer fields

Any change to the standard, mappings, entropy function, test vectors, or
reproducibility lock requires a Profile Change Proposal and a new profile
version.
