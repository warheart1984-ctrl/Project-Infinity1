# Sovereignty Boundary Specification

## Boundary Rule
All state transitions from non-sovereign actors into AAES OS sovereign state must pass through TVP and CEN.

## Sovereign Zone
- CEN
- TVP
- invariant registry
- sovereignty ledger
- law-of-laws ledger
- evidence receipts

## Non-Sovereign Zone
Application services, external actors, user-facing APIs, and orchestration layers.
